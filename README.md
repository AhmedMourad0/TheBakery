# The Bakery

## Project Overview
A baking app.

## Key Concepts
- ExoPlayer
- Home Screen Widgets
- Retrofit with Gson
- RxJava
- Fragment navigation pattern
- Room
- Reactive Event Bus
- BroadcastReceiver
- Picasso, ButterKnife
- Espresso Ui tests
